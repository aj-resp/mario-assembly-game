# 🎮 Mario Assembly Game

A Mario-inspired platformer developed in x86 MASM Assembly using the Irvine32 library. The game features multiple playable levels with unique environments, enemy AI, hidden mechanics, sound integration, and real-time gameplay systems — all implemented without using any game engine.

The project demonstrates low-level programming concepts such as memory handling, collision detection, keyboard input processing, file handling, and real-time game logic using pure Assembly language.

---

## 🚀 Features

- Multiple playable levels with different environments
- Hidden secret coin room accessible through pipe in level 2
- Enemy AI including Goombas, Koopas, and a boss enemy
- Boss enemy dynamically follows the player
- Coin magnet mechanic using the `M` ability
- Real-time movement and collision detection
- Moving platforms and lava hazards
- Fireballs and power-ups
- Persistent high-score saving using file handling
- Sound effects integrated using Windows sound API
- Main menu with load game and high-score system

---

## 🎮 Controls

- `A` / `D` → Move left and right
- `W` → Jump
- `M` → Collect nearby coins using magnet ability
- `S` / Down Key → Enter or exit secret pipe rooms

---

## 🛠️ Technologies Used

- x86 MASM Assembly
- Irvine32 Library
- Windows API (Sound Integration)

---

## ▶️ How to Run

1. Install MASM and Irvine32 setup
2. Open the project in Visual Studio
3. Assemble and run the program
